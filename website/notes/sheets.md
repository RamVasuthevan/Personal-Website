## Keyboard shortcuts
- New line in cell: <kbd>⌘</kbd> + <kbd>↩</kbd>
- Insert date: <kbd>⌘</kbd> + <kbd>;</kbd>
- Insert time: <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>;</kbd>
- Insert date and time: <kbd>⌥</kbd> + <kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>;</kbd>
- Insert row below: <kbd>⌃</kbd> + <kbd>⌥</kbd> + <kbd>I</kbd> then <kbd>R</kbd> then <kbd>B</kbd>
## Notes

1. You can only have one [table](https://support.google.com/docs/answer/14239833?hl=en) per sheet. [↗](https://www.reddit.com/r/googlesheets/comments/1dw868a/how_to_add_multiple_tables_in_one_sheet_prebuilt/)
