[Ghostty](https://ghostty.org/) is a terminal emulator

## CLI Commands

1. `ghostty +list-keybinds --default`
    - Lists the built-in default keybindings that ship with Ghostty
2. `ghostty +list-keybinds`
    - Lists your current active keybindings (defaults + your config overrides)

## Config

- `~/.config/ghostty/config`
    - Contains the configuration for Ghostty
- In keybind syntax, `super` = the Command key (⌘) on macOS
