Keyboard Shortcuts (for macOS)

- Toggle Database View: `⌘1`
- Refresh Database: `⌘R`

- [SQL generator](https://www.jetbrains.com/datagrip/features/generation.html#:~:text=SQL-,generator,-DDL%20statement%20for): Get DDL for one or more database objects
