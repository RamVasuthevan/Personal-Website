## Useful Keyboard Shortcuts

- New Tab: <kbd>⌘</kbd> + <kbd>T</kbd>
- Open Bookmark Manager: <kbd>⌥</kbd> + <kbd>⌘</kbd> + <kbd>B</kbd>
- Focus a specific tab: <kbd>⌘</kbd> + <kbd>1</kbd> to <kbd>⌘</kbd> + <kbd>8</kbd>

## [Disable JavaScript](https://developer.chrome.com/docs/devtools/javascript/disable)

1. Open DevTools
2. Open Command Menu (<kbd>⇧</kbd> + <kbd>⌘</kbd> + <kbd>P</kbd>)
3. Select `Disable JavaScript`
